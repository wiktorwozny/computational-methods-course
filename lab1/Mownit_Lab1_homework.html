  • Napisa program w jzyku Julia do obliczania iloczynu skalarnego wektorw (LinearAlgebra.dot) i mnoenia macierzy kwadratowej przez wektor z uyciem operatora *.
  • Uruchomi i zmierzy czasy dziaania obydwu funkcji - kadej dla rnych rozmiarw wektorw. Dokona 10 pomiarw dla kadego rozmiaru wektora.
  • Czasy dziaania powinny by zapisywane do jednego pliku CSV. Naley zaplanowa odpowiedni struktur kolumn takiego pliku.
  • Wczyta dane z w/w pliku do jednego DataFrame w jzyku Julia.
  • Korzystaj z mechanizmw DataFrame w jzyku Julia obliczy redni i odchylenie standardowe, w taki sposb, aby narysowa wykresy rednich czasw obliczenia operacji w zalenoci od rozmiaru wektora. Doda supki bdw do obydwu wykresw uzyskanych z obliczenia odchylenia standardowego.
  • Prosz poukadac wykresy w tabelk za pomoc opcji layouts: https://docs.juliaplots.org/latest/layouts/#Simple-Layouts-1.
  • Naley zadba o staranne podpisanie osi i wykresw.

function my_own_dot(x, y)
    sum = 0
    for i in 1:size(x, 1)
        sum += x[i] * y[i]
    end
       
    return sum
    
end

function matrix_times_vector_timetest(size)
    
    m = rand(0.0:0.01:1.0, size, size)  # random matrix size x size
    v = rand(size)
    return @elapsed matrix_times_vector(m, v)
